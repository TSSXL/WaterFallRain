插件使用-------------------------------------

1>引入Waterfall.js文件
2>调用waterfa.open()方法
3>参数
 	调用方式(参数可均可选)
  	waterfall.open({
 		text:'显示的文字',
 		fontSize:'字体的大小',
 		speed:'下落的速度',
 		preAction:true//是否显示预加载动画
 		color:'字体颜色',
 		fontFamily: '字体样式'
 	});
4>关闭瀑布雨
  waterfall.close();
